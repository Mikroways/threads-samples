# Qué hace este ejemplo

Crea archivos de 1.5GB en el /tmp. Para ello utilizará tantos threads como se
indique. Cada thread creará un archivo diferente.

## Como se corren estos ejemplos

## Python

Corremos directamente con:

```bash
python io-bound.py
```

## Ruby

Corremos directamente con:

```bash
ruby io-bound.rb
```

## Java

Primero compilar:

```bash
javac IoBound.java
```

Luego ejecutamos:

```bash
java IoBound
```
